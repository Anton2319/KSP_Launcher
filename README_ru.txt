Установка
---------------
Откройте файл launcher.jar, который вы только что скачали и создайте для него отдельную папку (НЕ КИДАТЬ В ПАПКУ С KSP!)
Если открыть не получается?
1. Вероятно что у вас не установлена Java, либо установлена устаревшая версия (для запуска лаунчера нужна версия java не ниже 11)
   скачать её можно по этой ссылке: https://www.oracle.com/technetwork/java/javase/downloads/jdk11-downloads-5066655.html
2. Jar-файл не помечен как исполняемый, перейдите в свойства и установите галочку в пункте исполняемость.
3. Всё не помогло? Попробуйте запустить лаунчер через cmd, для этого выполните следующее:
   ####
   Нажмите Win+R
   ####
   Введите cmd.exe
   ####
   В открывшейся консоли запустите:
	cd <Место где находится лаунчер>
   Например:
	cd C:\Users\anton\KSP_Launcher\
   ####
   Запустите команду:
	java -jar -Xms512M launcher.jar
   ####
